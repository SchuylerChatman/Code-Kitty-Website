GOAL

Goal

"Our end goal for Code Kitty is to create a block-based coding game that will teach the upper elementary/middle schoolers, teens, young adults, and hobbyists the fundamentals of programming. The game will consist of multiple different levels that will be tailered to each user's programming experience. From 'Hello World' to advanced data structures, Code Kitty will have something for everyone! Code Kitty strives to be both enertaining and engaging for all users. With the increasing need for computer literacy, growing computer science knowledge, and basic problem solving skills, Code Kitty aims to help provide the building blocks for anyone intersted in learning CS."

Whats our Approach?

Approach

"The tools we will use in our approach to accomplishing our goal consists of PlayCanvas and Blockly. As for user experience, users will play as a bionic cat. Users will be forced to navigate and hack their way through buildings secured by guards and cameras. Each hacking challenge will be represented as fundamental coding questions. An example scenario would be the user walking up to a locked gate to hack open. The user will then be prompted with the question: 'Print "Hello World!" to the screen.'. There would be multiple blocks with potential answers. Each wrong answer will deduct one life while the corrent answer gains access to what is stored behind the door."
